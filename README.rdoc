== ProductManagement

This application is for product management.

*Prerequisites*
* Ruby(2.6.2)
* Rails(5.2.3)
* redis-server

*Installation*

First clone the project repository from github.

* Install [rvm](https://rvm.io/rvm/install).
* Then go to project directory and run 'bundle install'
* Install [redi-server](https://www.digitalocean.com/community/tutorials/how-to-install-and-secure-redis-on-ubuntu-18-04)

*RunApplication*
* rails s
* redis-server
* bundle exce sidekiq
